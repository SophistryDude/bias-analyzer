# X Political Figures 500 — Continuation Plan

## Overview
Creating `docs/x-political-figures-500.md` — 500 profiles of the most discussed political figures on X (Twitter), using the same format as `docs/profile-breakdowns.md`.

## Format Per Profile
Each profile includes:
- Platform, Current Leaning, Ideological Coherence (0-1), Driver type
- Bias Trajectory table (2-4 rows showing evolution)
- Multi-Axis Position table (5 axes: Economic, Speech, Progressive, Liberal/Conservative, Foreign Policy)
- Notes section applying Logic System principles (epistemological classification, coherence analysis, category errors, dimensional preservation, constraint reduction)

## Logic System Principles to Apply
1. **Epistemological taxonomy**: Tag each figure's dominant rhetorical mode (do they present tacit understanding as known truth? value judgments as facts?)
2. **Coherence as constraint consistency**: High coherence = solvable puzzle (predict new positions from existing). Low coherence = external forces driving positions.
3. **Model vs Reality**: We don't decide who's right. We map positions and track consistency.
4. **Dimensional preservation**: Each axis scored independently. Never collapse to left-right.
5. **Progressive ≠ Liberal**: Load-bearing distinction. Score these axes independently.
6. **Category errors**: Note when figures routinely present one epistemological type as another.
7. **Persona divergence**: Note when public vs private positions diverge (Overton window manipulation).
